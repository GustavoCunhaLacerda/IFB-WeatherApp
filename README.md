## 💻 Como testar e utilizar

### Requisitos

**Clone o repositório e acesse a pasta**

```bash
$ git clone https://github.com/GustavoCunhaLacerda/IFB-WeatherApp.git && cd PC3TrabalhoFinal/frontend
```

**Siga esses passos**

```bash
# Install the dependencies
$ npm install

# Start the client
$ npm start
```
